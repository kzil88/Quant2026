# 均值回归策略原理与实现

## 核心思想

**涨多了会跌，跌多了会涨。** 价格终将回归到"正常水平"。

和动量策略完全相反——动量是追涨，均值回归是抄底。当一只股票跌得远低于其均值时买入，等它弹回来赚差价。

---

## 实现原理（3层信号叠加）

项目的均值回归不是简单比较"股价 vs 均线"，而是叠加了三个维度的判断（实现见 `quant2026/strategy/mean_reversion/strategy.py`）：

### 第1层：Z-score 偏离度（核心）

```
偏离度 = (收盘价 - MA) / MA          # 当前价格偏离均线多少
Z-score = (偏离度 - 偏离度均值) / 偏离度标准差   # 标准化
```

- Z-score < -1.0（默认阈值）→ 超卖，触发买入信号
- Z-score > +1.0 → 超买，触发卖出信号

**打分逻辑：score = -Z-score**（Z-score 越负，说明越超卖，得分越高）

### 第2层：布林带位置加分（可选）

计算股价在布林带中的位置（0=下轨，1=上轨）：

```
bb_pos = (收盘价 - 下轨) / (上轨 - 下轨)
bb_bonus = max(0, 1 - bb_pos)    # 越接近下轨，加分越多，最高+1
```

贴近布林带下轨 = 额外确认"确实跌过头了"。

### 第3层：RSI 超卖加分（可选）

```
if RSI < 30:
    rsi_bonus = (30 - RSI) / 30    # RSI=0时加1分，RSI=30时加0分
```

RSI < 30 是经典超卖信号，作为第三重确认。

### 最终得分

```
总分 = Z-score偏离分 + 布林带加分 + RSI加分
```

三个维度都说"这只股票跌过头了"时，得分最高，买入信心最强。

---

## 交易信号

| 信号 | 条件 | 含义 |
|------|------|------|
| **BUY** | Z-score < 阈值（默认 -1.0） | 严重超卖，买入 |
| **SELL** | Z-score > -阈值（即 > +1.0） | 超买，卖出 |
| **HOLD** | 介于两者之间 | 正常波动，不动 |

---

## 参数说明

| 参数 | 默认值 | 作用 |
|------|--------|------|
| `window` | 20 | 均线和标准差的回看窗口 |
| `zscore_threshold` | -1.0 | 超卖判定阈值，越小越严格 |
| `use_bollinger` | True | 是否启用布林带加分 |
| `use_rsi` | True | 是否启用 RSI 加分 |
| `rsi_period` | 14 | RSI 计算周期 |

---

## 适用场景与局限

**适合：** 震荡市、箱体行情——股价在一个范围内来回波动
**不适合：** 单边下跌（"跌多了会涨"在熊市里可能变成"跌多了还会跌"，抄底抄在半山腰）

关键风险：均值回归假设"均值是稳定的"。如果公司基本面恶化，股价下跌是合理的，不会回归。
