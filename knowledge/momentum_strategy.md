# 动量策略原理与实现

## 核心思想

**涨的还会涨，跌的还会跌。** 追踪价格趋势，买入正在上涨的股票，卖出正在下跌的。

学术上叫"动量效应"（Momentum Effect），是金融学里被研究最多的异象之一。直觉理解：市场信息扩散需要时间，一旦形成趋势，短期内大概率延续。

---

## 实现原理：双均线交叉 + 动量打分

项目中的动量策略不是简单看涨幅，而是用**双均线系统**（实现见 `quant2026/strategy/momentum/strategy.py`）：

### 1. 计算快慢均线

- **快均线（MA_fast）**：默认5日均线，反映短期趋势
- **慢均线（MA_slow）**：默认20日均线，反映中期趋势

```
MA_fast = 过去5天收盘价的平均值
MA_slow = 过去20天收盘价的平均值
```

### 2. 动量打分

```
score = (MA_fast - MA_slow) / MA_slow
```

含义：快均线比慢均线高多少（百分比）。

- score > 0：短期涨得比中期快，趋势向上，分越高动量越强
- score < 0：短期比中期弱，趋势向下
- score 越大：上涨趋势越猛

### 3. 交易信号（金叉/死叉）

通过比较**昨天和今天**快慢均线的位置关系：

| 信号 | 条件 | 含义 |
|------|------|------|
| **BUY（金叉）** | 昨天快线 ≤ 慢线，今天快线 > 慢线 | 短期趋势刚刚突破中期趋势，启动上涨 |
| **SELL（死叉）** | 昨天快线 ≥ 慢线，今天快线 < 慢线 | 短期趋势跌破中期趋势，开始走弱 |
| **HOLD** | 其他情况 | 趋势未变，维持现状 |

### 4. 选股逻辑

所有股票按 score 从高到低排序，取动量最强的若干只买入。

---

## 为什么用双均线而不是简单涨幅？

- **简单涨幅**（过去20天涨了多少）：容易被单日大涨大跌干扰
- **双均线**：平滑了价格波动，能更稳定地捕捉趋势方向
- **金叉/死叉**：提供明确的入场/出场时机，而不只是排名

---

## 参数说明

| 参数 | 默认值 | 作用 |
|------|--------|------|
| `fast_window` | 5 | 快均线周期，越小越灵敏，但假信号越多 |
| `slow_window` | 20 | 慢均线周期，越大越稳定，但反应越慢 |

经典组合：5/20（短线）、10/30（中线）、20/60（长线）

---

## 适用场景与局限

**适合：** 单边上涨/下跌的趋势行情，如牛市追涨
**不适合：** 震荡市（反复金叉死叉被来回打脸，俗称"左右挨耳光"）

通常和均值回归策略搭配使用——趋势行情用动量，震荡行情用均值回归。
